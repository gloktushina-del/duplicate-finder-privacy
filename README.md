# Политика конфиденциальности расширения Duplicate Finder

**Дата:** 20 января 2026 г.

## Какие данные собираются

- Расширение временно анализирует содержимое активной вкладки (DOM: текст, изображения, элементы) только для поиска дубликатов.
- **Не собираются** персональные данные, пароли, финансовая информация, история браузера.

## Как используются данные

- Обработка происходит локально в браузере пользователя.
- Данные не сохраняются после завершения анализа.
- Данные не передаются на внешние серверы и третьим лицам.

## Передача и хранение данных

- Данные не продаются и не используются в рекламных целях.
- Могут сохраняться только технические настройки расширения (например, предпочтения пользователя) в хранилище браузера.
